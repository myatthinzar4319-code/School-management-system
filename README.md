# School Management System

A web-based Learning Management System developed using PHP and MySQL.
The system supports separate roles for teachers and students.

## Features
- User authentication (Student & Teacher)
- Student registration and login
- Teacher dashboard
- Attendance marking and viewing
- Grade management
- Upload and view learning materials
- Image and file upload support

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- XAMPP
